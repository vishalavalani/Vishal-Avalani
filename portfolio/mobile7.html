SMB – Windows Phone 7 App

SMB App

As a client, Microsoft wanted to create Windows Phone app. App should be able to show Revenue Renewal Rate and Revenue Expiration $s for areas, subsidiaries and distributors. App displays Renewal Rate, Y/Y, Revenue Expired and Revenue Renewed for particular fiscal month. Data is fetched from SDF file which is uploaded on sky drive. App also shows 3 Month trending chart for Open Renewal %.

Technologies/Tools:

  • Visual Studio
  • .NET 3.5 C#
  • C#
  • XAML
  • Telerik Chart control
