Waypoint iOS App

AWP is a large consolidation of foodservice brokers that have joined together with one large retailer broker. They basically provide service to clients and customers in the foodservice industry. We created V1 app as POC using jQuery mobile which worked well. Hence, client wanted to convert it into native application using Xamarin so that most of the code can be shared between both apps. They also wanted to exploit smartphone functionalities and implement OpAware (Geofencing), Barcode Scanning, Voice Recording, Sample Express, Guarantee It etc which is an addition to V1 app.

Technologies/Tools:

  • Xamarin Studio
  • Xamarin iOS
  • C#
  • ASP .NET Web Api
